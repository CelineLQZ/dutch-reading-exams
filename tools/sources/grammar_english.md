# Dutch Word Order Differences — Exams 1–4 (4 Merged Types, English Explanations)

This file uses the merged 4-type grammar system. Each row contains: `Dutch sentence | Type | Word-order reason`.

## Grammar Type Reference — Beginner-Friendly English Version

| Type | Grammar rule | Beginner-friendly explanation | Typical pattern |
|---|---|---|---|
| **Type A — Verb before subject** | Finite verb before subject | Use this when you see **verb + subject**. It commonly happens in questions, after a fronted time/place/reason/object phrase, or after a fronted condition clause. | WH-word / yes-no verb / fronted element + fin. verb + subject + other parts |
| **Type B — Verb group at the end** | Subclause verb-final | In Dutch subclauses, the verb group often moves toward the end. English normally keeps the verb much earlier. | dat / als / wanneer / hoe / wat / die / of + subject + other parts + verb group |
| **Type C — Split verb / particle verb** | Separable verb / particle verb | Some Dutch verbs split: the main verb part appears early, and the small particle appears later. | verb part + other parts + particle, e.g. denk ... na, leg ... uit, haal ... op |
| **Type D — Dutch middle-position words** | Adverb / small-word placement | Dutch often puts time, place, manner, negation, and modal particles in the middle of the sentence. English often moves them elsewhere. | subject + verb + small adverb / negation / prep. phrase + other parts |
