# 荷兰语语序不一致句子整理 — Exams 1–4（4 类合并版，中文解释）

本文件使用 4 类合并后的语法分类。每行包含：`Dutch sentence | Type | 语序不一致点`。

## Grammar Type Reference — 中文小白版

| Type | Grammar rule | 小白版解释 | 典型结构 |
|---|---|---|---|
| **Type A — Verb before subject** | 变位动词在主语前 | 看到“动词 + 主语”时，优先考虑这一类。常见于问句、句首时间/地点/原因/宾语前置、或条件从句前置。 | 疑问词 / yes-no 问句动词 / 句首成分 + 变位动词 + 主语 + 其他成分 |
| **Type B — Verb group at the end** | 从句动词组后置 | 荷兰语从句里，动词组经常被放到从句后面。英文通常不会把动词拖到这么后。 | dat / als / wanneer / hoe / wat / die / of + 主语 + 其他成分 + 动词组 |
| **Type C — Split verb / particle verb** | 可分动词或小品词动词 | 有些荷兰语动词会拆开：前半部分放前面，小尾巴放后面。 | 动词前半部分 + 其他成分 + 小品词，如 denk ... na, leg ... uit, haal ... op |
| **Type D — Dutch middle-position words** | 小词/状语位置不同 | 荷兰语常把时间、地点、方式、否定词、语气小词放在句中；英文常会换位置。 | 主语 + 动词 + 小副词/状语/否定词/介词短语 + 其他成分 |
